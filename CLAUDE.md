# SimpleTrailer – Projektkontext

## Firma
- Name: SimpleTrailer
- Stadt: Bremen
- Domain: simpletrailer.de
- Produkt: PKW-Anhängervermietung (1-3 Anhänger)

## Ziel
Moderne Webseite für Anhängervermietung.
Referenz: RentMyTrailer.de – aber moderner und besser.
Prinzip: Wie Tier/Lime – Karte, Anhänger anklicken, buchen, bezahlen.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Datenbank: Supabase
- Zahlung: Stripe
- GPS: Teltonika + Traccar (kommt später)
- Hosting: Netlify
- Domain: simpletrailer.de

## Design
- Nutze immer den Frontend Design Skill für alle UI Entscheidungen
- Modern, clean, vertrauenswürdig
- Mobilfreundlich (mobile-first)
- Kein generisches AI-Design
- Bold und distinctive – SimpleTrailer soll auffallen
- Performance-optimiert (schnelle Ladezeiten)
